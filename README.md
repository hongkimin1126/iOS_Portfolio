# 자기소개서
안녕하세요. 서툴고 더디지만 꾸준히 성장하는 iOS 앱개발자가 되고 싶은 홍기민입니다.
저는 한국산업기술대학교(2013 ~ 2019) 메카트로닉스공학과를 전공하였습니다.
제가 iOS 앱 개발에 관심을 가지게 되었던 것은 대학교 2학년 재학 중 스타트업 창업한 사촌 형을 보고 새로운 시각을 접하게 되면서부터였습니다.
사촌 형의 두 번의 스타트업 창업(첫 번째 펀딩 기부 플랫폼: 위 제네레이션과, 두 번째 동영상 공유서비스(라이비오) 과정을 보게 되면서 모바일 개발자라는 직업이 운명처럼 다가왔습니다. 
가장 멋있게 생각해왔던 점은 개발에는 상한선이 없다는 것이었습니다. 보통의 업종과 다르게 일을 진행하면서 자기주장을 많이 반영할 수 있으며 본인 스스로가 제한만 두지 않는다면 무궁무진한 아이디어를 만들어 낼 수 있다는 부분과, 작은 개발 하나가 생활의 방식을 바꾸는 혁신 또한 제가 iOS 앱개발자가 되기로 결심한 이유 중에 하나였습니다. 
그렇게 본과 2학년 때부터 모바일 개발자에 관심을 가진 이후 최신동향을 지속적으로 추적해왔고 본과 졸업 이후 패스트캠퍼스라는 교육기관에서 6개월 iOS과정을 수료하면서 기본적인 Swift 지식을 습득하였습니다.
6개월 iOS 과정이 공대생이었지만 프로그래밍이 전공이 아니었던 저에게는 쉽지 않은 시간이었습니다. 
하지만 그럼에도 쉽게 포기하지는 않았습니다. 오래전부터 진중하게 생각하고 임한 것이기에 무거운 책임감으로 더 많이 노력해서 마지막에 수료할 때는 가장 많이 남겨가자는 각오를 다졌습니다.
그 결과 다른 전공자 동기들과 비슷한 실력까지 빠르게 따라갔던 경험이 있습니다. 저는 학습 초기에 스스로가 많이 부족한 것을 절실히 깨달았고 학원에 가장 일찍 나와서 가장 늦게까지 남아 학습을 하였던 것이 제가 빨리 발전할 수 있었던 이유라고 생각합니다.
본인의 부족함을 자각하고 느리지만 꾸준히 노력한다면 이루지 못할것은 없다고 생각합니다.
학원 과정 중에 몇 번의 간단한 프로젝트와, 백엔드 팀과의 REST API 실습, git과 노션을 통한 협업, 몇번 해커톤을 경험하였습니다. 또한 개인적으로 RXSwift 기초학습을 하였고 한 번의 AppStore에 App 배포 경험을 보유하고 있습니다.
